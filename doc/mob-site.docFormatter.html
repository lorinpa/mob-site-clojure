create-meta-keywords
(create-meta-keywords category-set)
generates an html meta-tag of keywords. 
parm category-set is a collection of terms derived from the RSS XML doc

webDocFormat
(webDocFormat docNode linkText href homeLinks)
generates a complete html document. 
parms:  
-docNode represents the document fragments derived from the RSS XML doc. 
-linkText and -href define a navigation link. 
-homeLinks sets whether we add links to the sites index.html
